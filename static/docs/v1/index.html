/users/ScratchCat

Paramaters

Field Type Details
nofollow None Doesn't get the user's followers and following
nostatus None Doesn't get the user's ocular status

Calling that will respond with a User Object

User Object

Field Type Details
username String The user's username
id Integer The user's id
about String The user's About Me
working_on String The user's What I'm Working On
status String The user's ocular status
status_color String The user's ocular color in a hex value
scratch_team Boolean If the user is on the sratch team
following Array of Partial User Objects The user's following
followers Array of Partial User Objects The user's followers
image_id Integer Used for getting the user's profile picture
country String The user's country

Partial User Object

Field Type Details
username String The user's username
